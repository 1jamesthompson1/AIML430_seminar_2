\namedframe{Amine}{Conclusion — Essentials \& highlights}{
\small
\begin{itemize}
  \item Remote sensing enables rapid situational awareness; satellites often \emph{lead} ground reports by 1–8h.
  \item Main limits: occlusion/latency (clouds, revisit), domain shift, and label scarcity/imbalance.
  \item Key results (ultra-brief): Fires—lightning 77.6\%; VIIRS $<$30\% all/$<$15\% lightning; monitorable $\downarrow$ with lightning ($r=-0.888$). Buildings—$\sim$91\% in/$\sim$80\% OOD; Providencia 97.5\%, $F_1=0.851$. Floods—YOLOv8 (synthetic) $\mathrm{mAP}_{50}=0.967$; field brittleness. Polar—96.5\% 4-way; 98.9\% target/no-target; $\sim$1\% false alarms. Wildfire tiles—ConvNeXt-S $\sim$99.05\%. ESRGAN lifts small/minority-class recall.
  \item Implication: When standardized and uncertainty-aware, systems speed triage and target aid; if not, blind spots (clouds/informal areas) $\Rightarrow$ inequity \& waste.
\end{itemize}
}

\namedframe{Amine}{Challenges \& next steps (compressed)}{
\small
\textbf{Challenges (realigned):}
\begin{itemize}
  \item \textit{Technical \& deployment:} edge compute/power/bandwidth; interpretability \& calibrated uncertainty.
  \item \textit{Data \& quality:} cost/access/latency; clouds/smoke; scarce/imbalanced labels.
  \item \textit{Operational \& methods:} cross-event/region generalization; multi-temporal reasoning; synthetic$\leftrightarrow$real gap.
\end{itemize}
\textbf{Next steps:}
\begin{itemize}
  \item Task$\to$action pipelines with latency/decision budgets (auditable).
  \item Distill/quantize ConvNeXt-S, YOLOv8n/s/m for robust edge use.
  \item SAR+optical+thermal+UAV fusion with ERA5 context to operate through clouds/smoke.
  \item Label-efficient learning + ESRGAN; standardized capture \& fair OOD/occlusion/latency evaluation.
\end{itemize}
\textbf{Bottom line:} earlier alerts, faster triage, safer navigation—delivered \emph{equitably} and with \emph{accountable} AI.
}
